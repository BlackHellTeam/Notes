###Xcode Choose Where to Open the New File
opt+shift+single-click

### Tab Move Codes in Xcode
command+[ for moving codes to left for a tab  
command+] for moving codes to right for a tab

### Code Completion in Xcode
esc to toggle

### Select Codes in Code Completion in Xcode
control+. for next  
shift+control+. for last

### Toggle Assistant Editor
opt+command+return to open  
command+return to remove

### Fold Code
opt+command+left-direction to fold  
opt+command+right-direction to unfold

### Open Edit Scheme
shift+command+,
